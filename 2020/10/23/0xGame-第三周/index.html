close_eyes

能够判断为盲注
要先猜测SQL语句：

select username from user where username='". $username ."' and password='". $password ."';

(condition)为假的时候，（比如我用户名输入1’ or 1=0#或者乱输一个用户名密码），页面回显“数据库里没你这号人,别想骗劳资.jpg”
为真的时候（比如我输入1’ or 1=1#）页面回显“数据库有你这号人，那又怎么着？”，根据回显情况的不同，我们就可以判断我们的condition究竟是真还是假。
附上官方wp二分法盲注脚本：

threeThousand

解3000层压缩包，每层都有两位数字密码，需要简单爆破一下
编写如下脚本:

import zipfile
import os
from time import time
def unZip(filename):
zipFile = zipfile.ZipFile(filename, 'r')
for i in range(100):
pwd = "0"*(2-len(str(i))) + str(i)
try:
zipFile.extractall(pwd=pwd.encode())
os.remove(filename)
break
except:
continue
st = time()
for i in range(3000):
namelist = os.listdir(os.getcwd())
for file in namelist:
if file[-4:] == ".zip":
unZip(file)
print(time()-st)
