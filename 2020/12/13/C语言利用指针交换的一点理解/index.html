C语言利用指针交换的一点理解

发表于 2020-12-13 分类于 C语言

晚上写实验报告，突然发现对利用指针交换的函数的理解完全不够，所以重新听了一遍翁恺老师的课来重新总结一下

首先理解一下数据在计算机中的存储：
先是定义，x在栈中被定义为int类型，能够在内存中开辟一个相应大小的空间，而x就指向这个空间
再对x进行赋值，也就是将1转化为二进制放入了那个空间里
而至于对指针的存储，如int* p
此时p是一个指向int类型的指针变量，而指针变量则一直是需要四个字节的空间来存储（即指针也有自己的地址
当需要对p赋值x的地址时，又要谈回x所开辟的空间
此时x为int，有四个字节的空间，每个字节对应有一个地址值，整个x的地址则是第一个字节的地址值
而指针类型的p就是能够通过&来获取x的地址
也就是说当p赋值以后，函数传入p便能够顺着x的地址去对x空间内的值进行操作

这时放上最经典的利用指针进行的简单交换

void swap3(int* a, int* b)
{
    int temp = *a;
    *a = *b;
    *b = temp;
}
int main()
{
    int x = 1, y = 2;
    swap3(&x, &y);  
    printf("x=%d,y=%d\n", x, y);
    return 0;
}

这样这个利用指针来交换数据的函数便不难理解了
函数swap内定义的只是int类型的temp
通过*来将a（即传入的&x）的值中的地址指向的数据传递给temp
而后再将a指向的数据空间的值更替为b指向的
最后再对b进行类似操作
那么这时候两个空间里的数值便发生了交换
也就是x和y的空间里的值发生了交换
所以再将x和y输出，就能够得到交换后的结果
