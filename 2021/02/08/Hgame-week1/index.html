helloRe

签到题没什么难的（虽然刚开始没看到自减给我带来了一定的困扰
拖进ida找到关键函数

即flag异或一个数与所给的加密后的文本相同
其中所给的文本显示为一堆乱码

根据曾经做的jj大佬出的题，判断出是需要按D转换为data

于是获得正确的数据，shift+e复制到脚本中解得flag

text = [151, 153, 156, 145, 158, 129, 145, 157, 155, 154,
        154, 171, 129, 151, 174, 128, 131, 143, 148, 137,
        153, 151]
flag = ''
key = 0xFF
for i in text:
    i ^= key
    key -= 1
    flag += chr(i)
print(flag)

raw_flag = input('give me your flag:\n')
text = list(raw_flag)
cipher = text[6:-1]
length = len(cipher)
for i in range(length//2):
    x = cipher[i*2+1]
    cipher[i*2+1] = cipher[i*2]
    cipher[i*2] = x
res = []
for i in range(length):
    res.append(ord(cipher[i]) ^ i)
res = bytes(res).hex()
print('your flag:'+res)

res = "30466633346f59213b4139794520572b45514d61583151576638643a"

flag = ""
for i in range(len(cipher)):
    x = ord(cipher[i]) ^ i
    cipher[i] = chr(x)
for i in range(len(cipher)//2):
    x = cipher[i*2+1]
    cipher[i*2+1] = cipher[i*2]
    cipher[i*2] = x
